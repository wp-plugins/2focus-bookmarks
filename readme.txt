== Description ==

Плагин позволяет читателям добавлять записи блога в Сервисы Закладок. Плагин полностью управляется из административной части. Есть возможность выбирать отображаемые сервисы.
В плагине реализована возможность использования тегов **rel=nofollow** и **noindex** (SEO часть). Два встроенных набора иконок и возможность настроек отображения через css файл плагина без вмешательства в код.
Так же Вы можете управлять местом отоброжением плагина, есть возможность выбрать - "на всех страницах", "только нас страницах записей", "на отдельных страницах"

== Installation ==

= Автоматическая установка =

Скопируйте папку с плагином в директорию `/wp-content/plugins/` и активируйте его в панели администрирования WordPress. Все:)


= Ручная вставка кода =

Если вы хотите вручную управлять отображением иконок, вставьте в свой шаблон следующий кусок кода: **<?php focus(); >** там, где вы хотите вывести иконки. Обратите внимание,
что этот код должен быть размещен внутри цикла **TheLoop**, т.е. между **<?php while (have_posts()) : the_post(); >** и **<?php endwhile; >**
Управлять внешним видом иконок можно через файл стилей **bookmark.css** в каталоге с плагином.
